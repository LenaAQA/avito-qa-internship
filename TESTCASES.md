# Тест-кейсы для API микросервиса объявлений

## Общая информация
- **Base URL**: https://qa-internship.avito.com
- **API версии**: v1
- 
## Описание API

| № | Метод  | Путь                   | Назначение                           |
|---|--------|------------------------|--------------------------------------|
| 1 | POST   | /api/1/item            | Создать объявление                   |
| 2 | GET    | /api/1/item/{id}       | Получить объявление по ID            |
| 3 | GET    | /api/1/{sellerId}/item | Получить все объявления продавца     |
| 4 | GET    | /api/1/statistic/{id}  | Получить статистику по объявлению    |

## Техники тест-дизайна
- **Классы эквивалентности** — валидация sellerId, price, name
- **Граничные значения** — числовые поля и длина строк
- **Error guessing** — нестандартные сценарии, XSS-инъекция
